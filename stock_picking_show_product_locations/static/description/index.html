Product Locations in Stock Picking

This module was written to extend the functionality of stock picking to support preview of product locations in stock picking form

Installation

To install this module, you need to:

  • upload module to your addons folder
  • restart odoo service
  • upgrade modules list
  • install as usual

Usage

To use this module, you need to:

  • Open `Internal Moves` record to see `Locations Of The Product` field
